**雑多なタブにさよなら。ウェブサイトのアクセスや検索をすばやく行えます。林澈.LYC タブはシンプルなUIと効率的な操作性を提供し、新しいタブページをスマートにします。**

## 🔹 主な機能

- **検索エンジン**：複数の検索エンジンを内蔵、キーワード検索もページ遷移不要。

- **ショートカット管理**：よく使うウェブサイトのアイコンをカスタマイズ、一クリックでアクセス可能。（コード修正必要）

- **動的背景**：動画や画像を背景に使用可能、高品質な視覚体験を提供。（追加もコードで可能）

- **オープンソース**：コードはオープン、テーマやスタイルのカスタマイズ可能、商用利用禁止。
