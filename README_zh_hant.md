## 🔹 核心功能

- **搜尋引擎**：內建多個搜尋引擎，可直接搜尋關鍵詞，無需跳轉到搜尋網站。

- **快捷方式管理**：自訂常用網站圖示，一鍵訪問，高效直達。（需修改程式碼）

- **動態背景**：可使用影片或圖片背景，營造高質感視覺體驗。（除預設外，可自行修改程式碼添加）
