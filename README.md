## 🔹 核心功能

- **搜索引擎**：内置多个搜索引擎，可直接搜索关键词，无需跳转到搜索网站。

- **快捷方式管理**：自定义常用网站图标，一键访问，高效直达。(有门槛,需修改代码)

- **动态背景**：可使用视频或图片背景，营造高质感视觉体验。  (除预设外,可自行修改代码添加)
