## 🔹 핵심 기능

- **검색 엔진**: 여러 검색 엔진 내장, 페이지 이동 없이 키워드 검색 가능.

- **단축키 관리**: 자주 사용하는 사이트 아이콘을 커스터마이즈, 클릭 한 번으로 접근 가능. (코드 수정 필요)

- **동적 배경**: 영상 또는 이미지 배경 지원, 고품질 시각 경험 제공. (코드로 추가 가능)

- **오픈 소스**: 코드 공개, 테마와 스타일 커스터마이즈 가능, 상업적 이용 금지.
