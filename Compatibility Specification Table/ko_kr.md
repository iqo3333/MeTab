# 호환성 사양

본 웹사이트는 **HTML5**, **CSS3**, **JavaScript (ES6+)**, 일부 고급 **CSS 애니메이션/레이아웃 기능**을 기반으로 구축되었습니다.

기능을 완전히 보장하는 최소 지원 브라우저 버전:

| 기능 모듈 | Chrome | Edge | Firefox | Safari | iOS Safari | Android Chrome | 비고 |
|-----------|:------:|:----:|:-------:|:------:|:----------:|:--------------:|------|
| 정밀 네트워크 시간 (`fetch` + `async/await`) | ✅ 42+ | ✅ 79+ | ✅ 39+ | ✅ 10+ | ✅ 10+ | ✅ 42+ | 네트워크 없으면 시스템 시간 사용 |
| 음력 계산 (JS) | ✅ 1+ | ✅ 12+ | ✅ 1+ | ✅ 3+ | ✅ 3+ | ✅ 42+ | 호환성 높음 |
| 시간 & 날짜 표시 (`Date` + `toLocaleTimeString`) | ✅ 46+ | ✅ 79+ | ✅ 29+ | ⚠️ 10+ | ⚠️ 10+ | ✅ 46+ | 구형 Safari는 표시 형식 다를 수 있음 |
| 검색 엔진 이동 (`window.open` + 정규식) | ✅ 1+ | ✅ 12+ | ✅ 1+ | ✅ 3+ | ✅ 3+ | ✅ 42+ | JS 필요 |

| 애니메이션 & 상호작용 (`requestAnimationFrame`) | ✅ 24+ | ✅ 79+ | ✅ 23+ | ✅ 6+ | ✅ 6+ | ✅ 42+ | 부드러운 갱신 보장 |

## 권장

1. **데스크탑**  
   - 추천: Chrome 100+, Edge 100+, Safari 15+, Firefox 100+  
   - 구버전도 동작하지만 애니메이션 지연, 네트워크 시간 정확도 저하 가능.

2. **모바일**  
   - 추천: iOS Safari 15+, Android Chrome 100+  
   - 영상 자동재생은 muted 필요.  
   - 구형 장치에서 핀치/더블탭 확대 방지 완벽 동작 안됨.

3. **비추천**  
   - IE 계열 (`classList.toggle` 애니메이션, `fetch`, `async/await` 미지원)  
   - 매우 구형 모바일 브라우저 (< iOS 10 / Android Chrome 50)  

⚠️ 참고: 본 표는 “Ver.9.1.2” 및 이전 버전에 적용되며, 향후 버전과 다를 수 있습니다.
